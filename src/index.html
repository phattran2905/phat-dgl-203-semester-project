Pokemon Land

Phat Tran

Description

This project is a simple game about pokemon in which the player finds and catches pokemon. The idea of this game is to explore the pokemon world. In particular, the player searches for available pokemon and decides whether to catch them or fight to earn rewards. Depending on what kind of pokemon it is, the catch process requires a different type of Pokeball. The rewards consist of various items that can help pokemon to increase their strength and level. In addition, the pokemon evolves if they reach a specific level and increase their stats.

My goals

Since DGL 203 is an advanced CSS class, my goal for this project is to practice the CSS skill with grid, flexbox, and animation. Moreover, I want to implement Sass to build this project.
